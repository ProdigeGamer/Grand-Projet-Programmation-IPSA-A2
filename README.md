*Par Skander et Lilian*

## C'est quoi ce projet ?

On a créé une app de messagerie sécurisée où les messages sont cryptés  
Y'a deux versions avec des méthodes de cryptage différentes.
Une version de cryptage forte et une version pédagogique 

---

## Comment lancer le code sur VSCode

### 1. **Ouvrir le projet**
- Ouvre VSCode
- File → Open Folder → Sélectionne le dossier "Grand-Projet-Programmation-IPSA-A2-Skander-Lilian"
- Dans le main file à gauche dans la barre deux version proposées : "Version Decalage ASCII sans UI" et "Version methode de Fernet avec UI"
- Il faut choisir une des deux version

## POUR LA VERSION DECALAGE ASCII SANS UI ##
- Clicker dans le fichier UI messagerie.py
- Clicker en à gauche sur lancer le code python sur un nouveau terminal
- Puis ouvrez un nouvel environement dans un termianl bash de vscode
- Lancer de nouveau le code dans le nouvel environement bash
- sur le 1er onglet crée une messagerie/entrée un port "5555" par exemple/ entre une valeur K / et copier les informations de connexion les coller dans un bloc note puis rentrer les informations dans l'utilisateur 2.

  ## POUR LA VERSION METHODE FERNET AVEC UI ##
-Ouvrez le terminal bash et entrée 
-pip install fernet
-pip install cryptography
- Clicker dans le fichier UI messagerie.py
- Clicker en à gauche sur lancer le code python sur un nouveau terminal
- Puis ouvrez un nouvel environement dans un termianl bash de vscode
- Lancer de nouveau le code dans le nouvel environement bash et suivre la même procédure que la methode asccii
